# **A simple calculator with Python programming language**

## *Usage* :
### *multiply and division operation* 
```bash
 >> 9 *2
     18
 >> 30/2
     15
```

### *addition and subtraction operations*
```bash
 >> 19+10
     29
 >> 40-25
     15
```

### *exponential operation*
```bash
 >> 9^2
     81
```
